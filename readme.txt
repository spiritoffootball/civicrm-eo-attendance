Attendance functionality for CiviCRM Event Organiser plugin.



== Description ==

This plugin provides attendance functionality for CiviCRM Event Organiser plugin.

This plugin requires at least WordPress 4.7 and CiviCRM 4.7.

It also requires:

* CiviCRM Event Organiser version 0.3 or greater

Be aware that this plugin is still in development. Test often, test thoroughly and open an issue if you find a problem.



== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
